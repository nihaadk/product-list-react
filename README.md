# Product list app

With this app add new Prodact and watch only product.

### Run the Application

Install all node_modules

```
npm install
```

Run on localhost:4200

```
ng start
```

## Built With

* [React](https://reactjs.org/)
* [Bootstrap](http://getbootstrap.com/)
